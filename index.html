📚 参考情報リンク

気象庁 地震情報 JMA

日本国内の地震情報をリアルタイムで提供。震度分布図や各地の震度一覧を確認できます。

気象庁 震度データベース JMA

過去の地震の震度データを検索可能。地点・期間を指定して詳細な震度情報を取得できます。

気象庁 震央分布図 JMA

日本周辺の震央分布を地図上で視覚的に確認できるインタラクティブマップです。

強震観測網 K-NET / KiK-net NIED

防災科学技術研究所の強震観測網。加速度波形データのダウンロードが可能（要ユーザー登録）。構造計算等に活用できます。

高感度地震観測網 Hi-net NIED

日本全国約800か所の高感度地震計で観測。連続波形データや自動処理震源リストを提供しています。

地震ハザードステーション J-SHIS NIED

全国の地震動予測地図や各地点の地震ハザード情報を閲覧可能。建築設計の参考資料として活用されています。

USGS 地震マップ USGS

アメリカ地質調査所の世界地震マップ。本アプリのデータソースです。リアルタイムの地震情報を提供しています。

USGS Earthquake API API

本アプリが利用しているAPI。GeoJSON形式で世界中の地震データを無料で取得可能。パブリックドメインです。

EMSC (欧州地中海地震学センター) EU

ヨーロッパ・地中海地域の地震情報を提供。体感報告や各国の地震速報も確認できます。

データ出典

地震データ: USGS (U.S. Geological Survey) - パブリックドメイン
地図タイル: OpenStreetMap contributors (ODbL)
地図ライブラリ: Leaflet (BSD-2-Clause)
